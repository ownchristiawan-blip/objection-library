ISM Objection Library

Internal sales tool for handling objections with structured responses,
smart search, and real-time analytics powered by Google Sheets.

------------------------------------------------------------------------

Overview

ISM Objection Library helps sales teams handle objections consistently
with structured responses, intelligent search, and usage analytics.

------------------------------------------------------------------------

Features

Smart Search

-   Exact match
-   Keyword match (synonyms)
-   Fuzzy match (typo tolerant)

Structured Responses

-   Soft
-   Medium
-   Hard

Analytics (By Period)

-   Date range filter
-   Top objections
-   Trend tracking

Logging System

-   Buffered logging
-   Batch sync
-   Timestamp tracking

------------------------------------------------------------------------

Tech Stack

-   Python
-   Streamlit
-   gspread
-   RapidFuzz

------------------------------------------------------------------------

Setup

Install dependencies

    pip install -r requirements.txt

Run app

    streamlit run app.py
